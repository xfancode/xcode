# **xCode v2.1**
[![Made by xFaN](https://img.shields.io/badge/Made%20by-xFaN-blueviolet)](https://github.com/xfancode)

> **Лёгкая, мощная, работает в браузере.**

## Возможности новой версии xCode

- 📁 **Добавление пользовательских файлов** — надоели обычные HTML/JS/CSS файлы? **Можно создать свои файлы!**
- 💾 **Автосохранение** — бывает такое когда пишешь код и случайно закрываешь вкладку? **Теперь при закрытии вкладки файл сохраняется! И не только HTML/CSS/JS, а ещё вами добавленные файлы!**
- 🗜️ **Экспорт в ZIP и HTML** — надоел обычный экспорт в HTML? **Добавлен экспорт не только в HTML, ну и ещё в ZIP!**
- 🖥️ **Запуск HTML** — теперь не надо экспортировать в один файл HTML. **Запусти HTML-файл прямо в xCode!**
- 🎨 **Material3** — теперь у вас есть не обычный IDE, а IDE с **Material3** что делает xCode красивым, а для вас - **удобным!**
- 🌐 **Работает без интернета** — сохрани `index.html` и пользуйся без интернета! **В этом и главная особенность xCode!**

## 🚀 Быстрый старт

1. Скачай xCode v2.1. Скачайте весь ZIP-архив xCod'а.
2. После скачивания разархивируйте всё в отдельную папку.
3. Открой `index.html` в любом браузере!
4. Напиши любой **код HTML**, и нажмите **Запустить**

> 💡 **Совет:** Используй xCode как стартовую страницу для своих веб-экспериментов.
